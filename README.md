# DSA Solutions

A comprehensive collection of Data Structures and Algorithms (DSA) problem solutions implemented in Python.

## Topics Covered

- Arrays
- Strings
- Linked Lists
- Stacks and Queues
- Trees and Graphs
- Sorting Algorithms
- Searching Algorithms
- Dynamic Programming
- Greedy Algorithms

## Structure

```
DSA-Solutions/
├── arrays/
├── strings/
├── linked_lists/
├── stacks_queues/
├── trees/
├── sorting/
├── searching/
└── dynamic_programming/
```

## How to Use

1. Navigate to the topic folder
2. Open the Python file for the specific problem
3. Review the solution and explanation
4. Run the code to test
